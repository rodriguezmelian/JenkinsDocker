Ahora vas a tu profile de gilLab https://gitlab-it.gscorp.ad/profile
En el panel izquierdo en SSH Keys
Aquí pondras la key generada anteriormente quedando asi

Finalmente le damos al botón "Add key".

La key privada deberas agregarla al store de Credenciales de Jenkins de la sgte. forma:
Ingresamos al JOB de nuestro Jenkins y nos vamos a la seccion "Configurar"
Alli en la seccion "Advanced Project Options" --> Pipeline encontramos que debemos configurar "Definition" en ese lugar seteamos ponemos "Pipeline script from SCM"
Esto nos habilita "SCM" donde elegimos "Git"
En "Repository URL" copiamos el path del proyecto Git en cuestion. Ej.:https://gitlab-it.gscorp.ad/JB01421/myProject
En "Credentials" vamos a añadir la key privada con el botón "add-->Jenkins"
Esto te mostrara un popup, donde tendras "Kind" deberas elegir "SSH user name with private key"
Luego seleccionamos el radio-button "Private Key" el cual te habilitara el input llamado "Key"
Aqui debemos volver a nuestro terminal linux/mac/gitBash Windows para copiarnos nuestra clave privada ejecutando este comando:
$ cat ~/.ssh/id_rsa
Copias la key que muestra el terminal al input "Key" de nuestro JOB Jenkins quedandote asi:

Finalmente botón "Add"
Con esto ya estariamos deberias tener autenticado tu proyecto gitLab.
